# 🤖 AI Agents

A collection of AI Agent projects built while learning **Agentic AI** using **Python, LangChain, LangGraph, and OpenRouter**.

This repository demonstrates the evolution of AI agents—from a simple single agent to autonomous and advanced multi-agent workflows.

---

## 🚀 Tech Stack

- Python
- LangChain
- LangGraph
- OpenRouter
- OpenAI Compatible APIs
- Pydantic
- Python Dotenv

---

# 📂 Projects

## ✅ 1. Single Agent

A basic AI agent capable of understanding user queries and generating intelligent responses.

**Concepts Covered**
- LLM Integration
- Prompt Engineering
- Agent Architecture

---

## ✅ 2. Planning Agent

An AI agent that first creates a plan before solving a problem.

**Concepts Covered**
- Task Planning
- Step-by-Step Reasoning
- Workflow Design

---

## ✅ 3. Coding Agent

A specialized AI agent focused on programming-related tasks.

**Concepts Covered**
- Code Generation
- Code Explanation
- Programming Assistance

---

## ✅ 4. Supervisor Agent

A supervisor agent responsible for delegating tasks to specialized agents.

**Concepts Covered**
- Task Routing
- Agent Coordination
- Orchestration

---

## ✅ 5. Multi-Agent System

Multiple AI agents collaborate together to solve complex tasks.

**Concepts Covered**
- Multi-Agent Collaboration
- Specialized Agents
- Agent Communication

---

## ✅ 6. Autonomous Agent

An AI agent capable of reasoning, selecting tools, and repeatedly executing them until the task is completed.

**Concepts Covered**
- Tool Calling
- Autonomous Reasoning
- Decision Making
- Tool Execution Loop

---

## ✅ 7. Advanced Agent

An enhanced AI agent combining planning, memory, reflection, and autonomous tool execution.

**Concepts Covered**
- Planning
- Memory
- Reflection
- Retry Mechanism
- Autonomous Tool Calling

> **Note:** This project is intended for learning Agentic AI concepts. Modern production systems generally implement these capabilities using LangGraph state management and checkpointing.

# 🎯 Purpose

The goal of this repository is to provide hands-on implementations of different AI agent architectures while understanding how modern autonomous systems are built.

Each project builds upon the previous one, gradually introducing more advanced agent capabilities.
